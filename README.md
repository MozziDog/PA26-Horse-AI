# Horse AI Research
## Overview
도로를 추종하고, 플레이어 입력에 반응하며, 장애물을 우회하는 말의 행동 AI를 구현한다.

## Reference
[The Legend of Zelda: Breath Of The Wild - Switch - Horse Riding](https://youtu.be/lq6MznRT5tI?si=edZSRRu8y7blf9po)

## Goals
AI
* 길을 따라 주행하는 상태가 아니라면 직진
* 길 가까이 있을 경우 길을 따라서 이동
* 플레이어의 입력에 따라 갈림길 선택 / 길에서 이탈
* 상황에 따라 장애물 우회 또는 점프로 극복
* 장애물 사이즈, 조향각 등 고려해서 극복 불가능하다면 rearing과 함께 멈춰서기

Movement
* 자연스러운 가속과 감속을 포함하는 물리 기반의 움직임
* 네 발의 위치를 고려해서 몸통의 회전 계산 (Suspension)
* 이동 불가능한 경사에서 미끄러짐

Debugging features
* 길의 node와 edge 시각화
* 말의 시선 등 정보 수집 시각화
* 현재 BT의 상태 시각화

## No-Goals
* '친밀도' 스탯에 따른 조작 반응성 변화 혹은 간헐적인 돌발 행동
* 말 탑승/하차 시스템
* Foot IK
